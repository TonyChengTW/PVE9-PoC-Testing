## 9. 結論與建議

### 9.1 核心發現

1. **xfs 優於 ext4**: 在所有 RAID 配置下，xfs 的延遲都比 ext4 低 40-50%

2. **RAID6+xfs 為最佳選擇**: 
   - 加權總分最高 (0.9286)
   - 延遲最低 (83.37 ms overall)
   - 利用率最佳 (61.09%)
   - 提供 2 磁碟損毀容忍度

3. **Block Size 影響顯著**: 16K block 在讀取測試中普遍表現較好

4. **效能與容量權衡**: 
   - RAID5/xfs 有較高利用率但延遲也較高；容量效率 87.5%
   - RAID10/ext4 利用率最低但延遲最高；容量效率僅 50%
   - RAID6/xfs 平衡點最佳
